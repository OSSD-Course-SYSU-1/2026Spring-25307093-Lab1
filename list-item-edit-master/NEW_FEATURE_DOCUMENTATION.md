# 新增功能说明文档

## 文档概述
本文档详细说明基于List组件的待办事项列表编辑应用的新增功能。该应用是一个完整的待办事项管理系统，具有宠物养成、番茄钟、多设备适配等特色功能。

## 项目基本信息
- **项目名称**: 待办事项列表编辑应用
- **技术栈**: HarmonyOS ArkTS
- **开发环境**: DevEco Studio 5.0.5 Release及以上
- **目标系统**: HarmonyOS 5.0.5 Release及以上
- **支持设备**: 华为手机、折叠屏、平板等

## 现有功能概览

### 核心功能
1. **待办事项管理**
   - 添加、编辑、删除待办事项
   - 标记任务完成状态
   - 设置任务优先级（低、中、高）
   - 设置任务截止日期
   - 左滑删除功能

2. **数据持久化**
   - 使用分布式KVStore存储数据
   - 支持跨设备数据同步
   - 本地数据备份

3. **宠物养成系统**
   - 虚拟宠物成长机制
   - 完成任务获得经验值
   - 宠物心情系统
   - 宠物喂食功能

4. **番茄钟功能**
   - 专注计时器
   - 休息计时器
   - 任务统计

5. **多设备适配**
   - 响应式布局设计
   - 支持手机、折叠屏、平板
   - 自适应字体大小

6. **个性化设置**
   - 深色/浅色主题切换
   - 字体大小调整
   - 数据迁移功能

## 新增功能详细说明

### 1. 宠物养成系统

#### 功能描述
集成虚拟宠物养成机制，将完成任务与宠物成长相结合，增加应用的趣味性和用户粘性。

#### 核心特性
- **宠物成长**: 完成任务可获得经验值，宠物随经验值增长而进化
- **心情系统**: 宠物心情随时间变化，完成任务可提升心情值
- **喂食功能**: 可定期喂食宠物维持其健康状态
- **互动动画**: 宠物有简单的动画效果和交互反馈

#### 技术实现
- 使用单独的KVStore存储宠物数据
- 宠物状态包括：名称、阶段、经验值、心情值、最后喂食时间
- 完成任务时自动更新宠物经验值
- 定期检查宠物心情衰减

### 2. 番茄钟专注功能

#### 功能描述
提供番茄工作法计时器，帮助用户提高工作效率。

#### 核心特性
- **专注计时**: 25分钟专注工作计时
- **休息计时**: 5分钟短休息计时
- **长休息计时**: 25分钟长休息计时
- **任务统计**: 记录完成的番茄钟数量
- **通知提醒**: 计时结束时的系统通知

#### 技术实现
- 独立页面管理番茄钟状态
- 使用定时器和状态管理
- 系统通知集成
- 数据持久化存储

### 3. 多设备适配系统

#### 功能描述
实现响应式布局，适配不同屏幕尺寸的设备。

#### 核心特性
- **断点系统**: 定义XS、SM、MD、LG四个断点
- **自适应布局**: 根据屏幕宽度自动调整布局
- **字体缩放**: 不同设备尺寸下的字体大小优化
- **网格布局**: 平板设备上的多列显示

#### 技术实现
- 监听屏幕尺寸变化
- 根据断点动态调整UI组件
- 使用Grid组件实现多列布局
- 响应式字体大小计算

### 4. 数据迁移功能

#### 功能描述
支持应用间数据迁移，方便用户更换设备或重新安装应用。

#### 核心特性
- **无缝迁移**: 自动检测并迁移待办事项数据
- **状态保留**: 迁移过滤状态和设置
- **用户提示**: 迁移完成后的友好提示
- **容错处理**: 迁移失败时的错误处理

#### 技术实现
- 使用AppStorage进行临时数据存储
- 实现onContinue生命周期方法
- JSON序列化/反序列化
- 异常处理和用户反馈

## 技术架构

### 1. 数据层
- **分布式KVStore**: 用于本地数据存储和跨设备同步
- **AppStorage**: 用于应用状态管理和数据迁移
- **内存缓存**: 运行时数据管理

### 2. 业务逻辑层
- **状态管理**: 使用@State、@Prop、@Link装饰器
- **事件处理**: 用户交互事件处理
- **业务规则**: 任务管理、宠物成长、番茄钟等业务逻辑

### 3. 视图层
- **组件化设计**: 模块化的UI组件
- **响应式布局**: 自适应不同设备
- **动画效果**: 平滑的过渡动画

### 4. 持久化层
- **KVStore配置**: 加密选项、同步策略
- **数据模型**: TodoItem、PetData等数据模型
- **序列化**: JSON格式的数据序列化

## 使用说明

### 1. 待办事项管理
1. **添加任务**: 点击底部输入框，输入任务内容，选择优先级和截止日期，点击添加按钮
2. **完成任务**: 点击任务左侧的复选框标记为完成
3. **编辑任务**: 点击任务内容进入编辑模式
4. **删除任务**: 左滑任务项，点击删除按钮

### 2. 宠物养成
1. **查看宠物**: 在主页查看宠物状态
2. **获取经验**: 完成任务可增加宠物经验值
3. **喂食宠物**: 点击宠物进行喂食，提升心情值
4. **宠物进化**: 经验值达到一定阈值后宠物会进化

### 3. 番茄钟使用
1. **启动番茄钟**: 切换到番茄钟页面
2. **开始专注**: 点击开始按钮开始25分钟专注计时
3. **休息时间**: 专注结束后自动进入休息计时
4. **查看统计**: 查看已完成的番茄钟数量

### 4. 个性化设置
1. **主题切换**: 点击右上角设置按钮切换深色/浅色主题
2. **字体调整**: 在设置中调整字体大小
3. **数据迁移**: 更换设备时自动迁移数据
